安裝pip

到https://bootstrap.pypa.io/get-pip.py頁面，滑鼠右鍵另存新檔到CD2020/data執行

1
python get-pip.py

安裝 CMSiMDE 所需模組

利用 pip 安裝 flask, bs4, lxml, pelican, markdown, flask_cors, leo (可單獨安裝區少的模組)

1
pip install flask bs4 lxml pelican markdown flask_cors leo

重新安裝Leo

解除安裝現有Leo(6.1版本)

1
pip uninstall leo

安裝Leo 6.2.1版本

1
pip install leo==6.2.1
